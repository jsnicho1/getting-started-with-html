Fundamentals of the Internet:

The Internet is a global network of interconnected computers. Computers make file requests via web servers through the http protocol.The server then sends the requested document or file back to the user. Browsers then render the webpage by dispaying the underlying code, most likely html.

HTML:

HTML stands for hyper text markup language. It is the basic "language" of the web. HTML is made of text and tags which together form the content and appearance of a webpage (along with other coding technologies). Web Developers use HTML to construct web pages by programming the computer to execute the exact instructions given to it.
